Language Symbol Example

Python Tips:

Ctrl-E : go to line end
Ctrl-K : clear line from cursor to end

Java

Writing a method in Java

    access return methodName(parameters){
        //code here
    }

access can be:

  • public
  • private
  • protected

return can be:

  • void
  • [any built-in type]
  • [any class type]
  • [any interface type]
